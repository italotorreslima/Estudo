*docker image pull mysql:5.6 - baixa a imagem do mysql na versão determinada
*docker container run myslq:5.6 - rodar o container (nao vai funcionar pq precisa das variaveis de ambiente)
** -e significa que posteriormente viram as variaveis de ambiente para o docker
*docker run -e MYSQL_ROOT_PASSWORD=root --name meu-mysql -d -p 3306:3306 mysql:5.6 - para rodar o container e definir uma senha que no caso será root e também o nome do bd e tambem refletir o localhost no container do host
*docker images - para ver a listagem das imagens
*docker ps - lista todos os containers que estao rodando
*docker stop {nome ou id do container} - ele finaliza o container
*docker rm {nome ou id do container} - para remover um container
*docker inspect {nome ou id do container} - lista parametros e propriedades do container
